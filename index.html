SHA256 Hash


A hash function is any function that can be used to map data of arbitrary size to data of fixed size. The values returned by a hash function are called hash values, hash codes, digests, or simply hashes. A very small change in the input value to the hash (even a bit change) causes the hash value to vary by a very large value. The hash functions that we use in cryptography are mainly one-way functions. This means that it is almost impossible to obtain the input from the hash value by trying to reverse the hash function.
